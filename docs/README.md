# SmartFranklin

SmartFranklin is a modular IoT firmware platform for the M5Stack Stick C Plus2.

Features:
- M5Unified framework
- PaHub I2C multiplexer
- Ultrasonic, Weight (U180 I2C NAU7802), MPU6050 sensors
- BM8563 RTC
- Xiaoxiang/JBD BMS via BLE
- Meshtastic C6L LoRa bridge
- NB-IoT2 cellular module
- Embedded MQTT client + optional external broker
- Dual-broker MQTT bridge (internal/external)
- QoS 1, retained messages, loop detection, prefixes
- Web dashboard + configuration portal
- OTA firmware updates
- Watchdog + fault recovery
- Plugin system for new sensors
